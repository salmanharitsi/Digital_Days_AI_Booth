User Requirement

  • Adanya fitur untuk sorting, search dan edit data
  • Grafik interaktif​
  • List keseluruhan data aspek dampak lingkungan PHR WK Rokan
  • Summary jumlah aspek dampak Lingkungan per wilayah operasi​
  • Summary jumlah tiap aktivitas, aspek, dampak Lingkungan disertai tingkat risiko nya
  • Overview Top Risk aspek dampak , sisa risiko​

Benefit

  • Mudah diakses oleh user
  • Menampilkan data aspek dampak Lingkungan secara ringkas, cepat, dan real time
  • Mudah memperbarui data
  • Mengurangi pekerjaan manual secara signifikan untuk memperbarui data

Background

kondisi saat ini:

  • Dokumen aspek dampak (Asdam) register disimpan dalam sharefolder internal OE/HES dan copy dokumen disimpan masing-masing user sehingga yang digunakan berpotensi bukan yang terupdate
  • Dokumen asdam dikelola secara manual sedangkan jumlah data sangat banyak dan merupakan live document yang dapat diperbarui tiap adanya perubahan aktivitas/kegiatan
  • Belum ada sistem untuk tracking pemantauan control tambahan dari asdam aktivitas/kegiatan

Solution

  • Mengembangkan dashboard PowerBi untuk menyajikan data aspek dampak (Asdam) register yang dapat diakses oleh seluruh user / facility owner untuk mempermudah akses, sosialisasi, dan ownership
  • Adanya dashboard dapat menyajikan kumpulan data dalam bentuk visual, melakukan analisis data dan laporan
  • Dashboard dapat diperbarui secara terus menerus jika adanya perubahan aktivitas/kegiatan yang berdampak pada Lingkungan dari kajian envid yang telah dilakukan
