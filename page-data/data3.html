User Requirement

  • Otomatisasi penarikan data dari beberapa database untuk ditampilkan pada OE/HES Dashboard – PHR War Room
  • Data dapat diperbarui secara otomatis setiap hari​
  • Data insiden dapat di-drill thru untuk menampilkan deskripsi
  • Dashboard dapat diakses dari ponsel​

Benefit

  • Mengurangi pekerjaan manual secara signifikan untuk memperbarui dashboard
  • Adanya penarikan data yang lebih reliable karena menggunakan data langsung dari database meliputi data insiden, manpower, man hours, safe man hours, dan UA/UC

Background

Kondisi OE/HES Dashboard saat ini:

  • Dashboard diperbarui secara manual setiap pagi hari oleh PIC sehingga status dashboard up to date pada pukul 07:30
  • Data yang dimasukkan secara manual termasuk data insiden, manpower, man hours, safe man hours, UA/UC, dan flash news

Solution

  • Melakukan integrasi antara dashboard dengan beberapa database sehingga pekerjaan manual untuk memperbarui dashboard setiap pagi hari dapat diminimalkan
  • Penarikan data otomatis meliputi:
  • Data insiden diambil dari impact QRA
  • Manpower, man hours, safe man hours, UA/UC diambil dari HOPES
  • Grafik MWT diambil dari MS Form
  • Grafik cumulative rate merupakan hasil pengolahan otomatis
  • OE/HES Flash News diambil dari SharePoint. Pada kondisi tertentu, PIC tidak perlu memperbarui status insiden namun informasinya dapat ditampilkan secara otomatis pada dashboard
  • Dashboard dapat diakses oleh pekerja melalui ponsel dengan penyimpanan di cloud
