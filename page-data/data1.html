User Requirement

  • Visualisasi progres dan status siklus proses CHESM dan action item CHESM
  • Otomatisasi penarikan data dari beberapa sumber (aplikasi dan spreadsheet)
  • Pemutakhiran data secara otomatis setiap hari

Background

  • Belum ada visualisasi status dan progres siklus proses CHESM dan action item CHESM
  • Konsolidasi dan pemantauan data status dan progres dilakukan secara manual menggunakan offline spreadsheet

The Winning Team

  • CHESM: M. Aditya Pradana, Misbahul Zakiyah, Mariska Lestari
  • IT: Agung Subarkat, Bayudita, Sinta Dewi

Benefit

  • Memberikan informasi kepada pelanggan (customer) mengenai status dan progress proses CHESM
  • Mengurangi pekerjaan manual secara signifikan untuk Konsolidasi dan pemantauan data dan mengurangi kesalahan informasi
  • Memberikan informasi awal mengenai siklus proses CHESM yang harus ditindaklanjuti sehingga tim CHESM memiliki perencanaan dan pengaturan resource yang lebih untuk memenuhi kebutuhan pelanggan (customer)

Solution

  • Membuat dashboard yang mengintegrasikan beberapa sumber data (aplikasi dan spreadsheet) untuk menampilkan 2 halaman, yaitu "Progres dan status siklus proses CHESM" dan "Action item CHESM"
  • Progres dan status siklus proses CHESM ditampilkan dalam circle dan bar chart dengan kemampuan untuk menampilkan informasi detil dalam tabel
  • Penarikan data untuk siklus proses CHESM dan action item CHESM dilakukan secara otomatis sehingga PIC Tim CHESM tidak perlu melakukan pemutakhiran secara manual di dalam dashboard
  • Data yang ditampilkan difilter untuk menghilangkan informasi yang confidential dan sensitive
