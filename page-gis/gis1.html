Background

Start Work Check (SWC) is a mandatory activity to ensure all preparations meet safety factors before the work begins. In the construction process at PHR WK Rokan, the SWC process is assisted by the JPK (Jasa Penyedia Konstruksi). PHR requires that the SWC process be carried out on time, obtain information on the number of workers and heavy equipment being used, and get information/feedback for any work constraints in the field quickly.

User Requirement

  • An integrated system to monitor :
    • When the SWC started
    • How many workers are involved
    • How many heavy equipment are utilized
    • Any obstacles occurred in the field
  • A system that can be operated by any perwira and mitra kerja related to this job;
  • A mobile application that can input and store data in conditions of no signal;

Solution

  • Data integration with FED Database, as prepopulated data for dropdown list in mobile app form : PMCoW Company, Project Name, Crew, and PMCoW;
  • Mobile application Daily SWC Monitoring Form using ArcGIS Survey123 : Data entry tool for surveyor in the field;
  • Reporting Dashboard using ArcGIS Dashboard:
  • SWC Monitoring Users Registration : To monitor number of registered users and number of surveys per company;
  • Dashboard SWC Monitoring Tool : To monitor SWC activities real time, ensure the form submitted on time, recognize number of HE and crew operating in the field
